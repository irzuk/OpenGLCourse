# ITMO MSE OpenGL homework template 2022
управление камерой:
W - верх
S - низ
A - лево
D - право
E - отдалить
F - приблазить

Модель: стандартная обезьянка из Blender, с натянутой текстурой из какой-то картинки.
Освещение Фонга немного с фиолетовым оттенком.
Трансформация - морфинг в сферу.
